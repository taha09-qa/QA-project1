================================================================
  1. FUNCTIONAL TESTING
================================================================

1.1 Login
-------------------------------------------------
  - Verify user can log in with valid credentials (standard_user / secret_sauce)
  - Verify login fails with an incorrect password and shows an error message
  - Verify login fails with an incorrect username and shows an error message
  - Verify submitting empty fields shows a validation error
  - Verify locked_out_user sees the correct locked-out error message
  - Verify all login page elements are visible on load (username, password, button)


1.2 Product Listing Page
-------------------------------------------------
  - Verify all 6 products display after login with name, price, image, and Add to Cart button
  - Verify sorting by price low-to-high reorders products correctly
  - Verify sorting by price high-to-low reorders products correctly
  - Verify sorting by name A-Z reorders products correctly
  - Verify clicking a product name opens the correct product detail page
  - Verify product detail page shows name, description, price, and Add to Cart button


1.3 Shopping Cart
-------------------------------------------------
  - Verify clicking Add to Cart adds the item and updates the cart badge count
  - Verify the Add to Cart button changes to Remove after item is added
  - Verify clicking Remove removes the item and resets the badge count
  - Verify cart page shows the correct item name and price after adding
  - Verify navigating to cart without adding items shows an empty cart
  - Verify Continue Shopping button returns the user to the product listing page


1.4 Checkout
-------------------------------------------------
  - Verify clicking Checkout from cart navigates to the checkout info page
  - Verify submitting checkout without first name shows a required field error
  - Verify submitting checkout without last name shows a required field error
  - Verify submitting checkout without zip code shows a required field error
  - Verify filling all fields and clicking Continue goes to the order summary page
  - Verify the order summary shows the correct item, price, tax, and total
  - Verify clicking Finish completes the order and shows a confirmation message
  - Verify clicking Cancel on checkout step 1 returns the user to the cart


1.5 Logout
-------------------------------------------------
  - Verify the burger menu opens and shows a Logout option
  - Verify clicking Logout redirects the user to the login page
  - Verify a logged-out user cannot access /inventory.html directly (redirected to login)
